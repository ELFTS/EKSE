## 📝 软件介绍

万物键盘音效是一款现代化的WPF键盘音效软件，为您的打字过程增添愉悦的声音体验。通过可视化界面，您可以轻松为每个按键设置不同的音效，创建属于自己的个性化键盘音效方案。

### 主要功能

- 🎵 **多格式音频支持**：WAV、MP3、AAC、WMA、FLAC等多种音频格式
- ⌨️ **可视化虚拟键盘**：直观的键盘界面，方便为每个按键设置音效
- 🎨 **多种声音方案**：支持创建和切换多个音效配置文件
- 📁 **音效文件管理**：添加、删除、重命名音频文件，轻松管理您的音效库
- 📤 **方案导出导入**：支持导出和导入声音方案，便于分享和备份
- ⚙️ **个性化设置**：音量调节、启动选项、窗口动画等多种设置选项
- 🎭 **现代UI设计**：基于Material Design的美观界面
- 📊 **系统托盘支持**：最小化到系统托盘，方便随时访问
- 🔄 **按键状态监控**：实时监控键盘按键状态，确保音效准确播放
- 🚀 **音效缓存优化**：智能缓存音效文件，提升播放性能

## 📦 安装

1. 从GitHub Releases页面下载最新版本的安装包
2. 双击安装包，按照提示完成安装
3. 安装完成后，即可启动应用程序开始使用

## 🚀 快速开始

1. 启动应用程序
2. 在主界面选择或创建一个声音方案
3. 点击虚拟键盘上的按键，为其选择音效文件
4. 开始打字，享受个性化的键盘音效体验

## 💻 开发环境

- Visual Studio 2022 (17.12 或更高版本)
- .NET 9 SDK

### 如何构建项目

1. 克隆仓库：`git clone https://github.com/ELFTS/EKSE.git`
2. 打开 `EKSE.slnx` 文件
3. 选择 "Release" 配置
4. 构建解决方案：`Ctrl+Shift+B`
